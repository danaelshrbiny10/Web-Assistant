## Installing 

To install **Web Assistant**, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/danaelshrbiny10/Web-Assistant.git
```

Windows:

```bash
git clone https://github.com/danaelshrbiny10/Web-Assistant.git
```
